## Lesson 1: Introduction to HTML - Building Blocks of the Web

### üéØ Learning Objectives
- Understand what HTML is and its role in web development
- Learn basic HTML document structure
- Create headings, paragraphs, and lists

### üìñ Topics Covered
- What is HTML? (HyperText Markup Language)
- HTML document structure: `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`
- Essential tags: headings (`<h1>` - `<h6>`), paragraphs (`<p>`), line breaks (`<br>`)
- Lists: ordered (`<ol>`) and unordered (`<ul>`)
- Comments in HTML (`<!-- -->`)

### üõ†Ô∏è Hands-On Project
Create a simple "About Me" page with a title, several paragraphs about yourself, and a list of your hobbies.

### üí° Key Takeaway

> HTML provides the structure and content of web pages, like the skeleton of a building.

---

## Lesson 2: HTML Links, Images, and Semantic Structure

### üéØ Learning Objectives
- Add links and images to web pages
- Understand semantic HTML elements
- Organize content with proper structure

### üìñ Topics Covered
- Links: anchor tags (`<a href="">`)
- Images: `<img src="" alt="">`
- Semantic HTML: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- Divs and spans for grouping content
- Attributes and their importance

### üõ†Ô∏è Hands-On Project
Expand your "About Me" page: add a profile picture, links to your social media or favorite websites, and reorganize content using semantic HTML tags.

### üí° Key Takeaway

> Semantic HTML makes your code more readable and accessible, helping both humans and search engines understand your content.

## Lesson 3: Introduction to CSS - Styling Your Web Pages

### üéØ Learning Objectives
- Understand what CSS is and how it works with HTML
- Learn different ways to add CSS to HTML
- Apply basic styling properties

### üìñ Topics Covered
- What is CSS? (Cascading Style Sheets)
- Three ways to add CSS: inline, internal (`<style>`), and external (`.css` file)
- CSS selectors: element, class (`.`), and ID (`#`)
- Basic properties: `color`, `background-color`, `font-size`, `font-family`, `text-align`
- The CSS box model introduction: `margin`, `padding`, `border`

### üõ†Ô∏è Hands-On Project
Create an external stylesheet and style your "About Me" page with colors, fonts, spacing, and borders.

### üí° Key Takeaway

> CSS is responsible for the visual presentation of your web pages, like paint and decoration for your building.

## Lesson 4: CSS Layout and Positioning

### üéØ Learning Objectives
- Master the CSS box model
- Understand layout techniques
- Position elements on the page

### üìñ Topics Covered
- Deep dive into the box model: content, padding, border, margin
- `display` property: `block`, `inline`, `inline-block`, `none`
- Basic positioning: `static`, `relative`, `absolute`, `fixed`
- Introduction to Flexbox for simple layouts
- Width and height properties

### üõ†Ô∏è Hands-On Project
Create a simple two-column layout for a personal portfolio page with a sidebar and main content area.

### üí° Key Takeaway

> Understanding layout is crucial for creating visually organized and responsive web pages.

## Lesson 5: Introduction to JavaScript - Making Pages Interactive

### üéØ Learning Objectives
- Understand what JavaScript is and its role
- Learn basic JavaScript syntax
- Manipulate HTML elements with JavaScript

### üìñ Topics Covered
- What is JavaScript? Client-side scripting
- Variables: `let`, `const`, `var`
- Data types: strings, numbers, booleans, arrays
- Basic operators and expressions
- `console.log()` for debugging
- Selecting HTML elements: `document.getElementById()`, `document.querySelector()`
- Changing content: `.innerHTML`, `.textContent`

### üõ†Ô∏è Hands-On Project
Add a button to your page that changes the text of a heading when clicked.

### üí° Key Takeaway

> JavaScript brings your web pages to life by adding interactivity and dynamic behavior.

## Lesson 6: JavaScript Functions and Events

### üéØ Learning Objectives
- Create and use functions
- Respond to user interactions with events
- Build interactive features

### üìñ Topics Covered
- Functions: declaration, parameters, return values
- Event listeners: `addEventListener()`
- Common events: `click`, `mouseover`, `mouseout`, `keypress`
- Arrow functions (modern syntax)
- Changing CSS with JavaScript: `.style` property
- Adding/removing classes: `.classList.add()`, `.classList.remove()`, `.classList.toggle()`

### üõ†Ô∏è Hands-On Project
Create an interactive color changer where users can click buttons to change the background color of the page.

### üí° Key Takeaway

> Functions and events are the foundation of interactive web applications.

## Lesson 7: JavaScript Control Flow and DOM Manipulation

### üéØ Learning Objectives
- Make decisions with conditional statements
- Repeat actions with loops
- Create and modify HTML elements dynamically

### üìñ Topics Covered
- Conditional statements: `if`, `else if`, `else`
- Comparison operators: `===`, `!==`, `>`, `<`, `>=`, `<=`
- Logical operators: `&&`, `||`, `!`
- Loops: `for` loop, `while` loop
- Array methods: `.forEach()`, `.map()`
- Creating elements: `document.createElement()`
- Appending elements: `.appendChild()`

### üõ†Ô∏è Hands-On Project
Build a simple to-do list application where users can add items to a list dynamically.

### üí° Key Takeaway

> Control flow and DOM manipulation allow you to create dynamic, data-driven web applications.

## Lesson 8: Capstone Project - Build a Complete Interactive Website

### üéØ Learning Objectives
- Combine HTML, CSS, and JavaScript skills
- Plan and execute a complete project
- Practice problem-solving and debugging

### üìñ Topics Covered
- Project planning and structure
- Responsive design basics: media queries
- Form elements: `<input>`, `<textarea>`, `<button>`, `<form>`
- Form validation with JavaScript
- Local storage (bonus topic): saving data in the browser
- Best practices: code organization, commenting, naming conventions

### üõ†Ô∏è Hands-On Project

**Choose one:**

1. **Interactive Quiz Application** - Multiple-choice questions with score tracking
2. **Personal Portfolio Website** - Showcase projects with filtering functionality
3. **Simple Calculator** - Perform basic math operations with a visual interface
4. **Weather Dashboard** - Display weather information with user input (can use mock data)

### üí° Key Takeaway

> Real projects are the best way to solidify your learning and build confidence as a developer.

## üë®‚Äçüè´ Teaching Tips

1. **Live Coding** - Demonstrate concepts by coding in real-time with students following along
2. **Encourage Experimentation** - Let students break things and fix them to learn debugging
3. **Provide Challenges** - Offer stretch goals for advanced students who finish early
4. **Code Reviews** - Have students share and discuss their projects with peers
5. **Debugging Practice** - Teach students to use browser developer tools early
6. **Build in Public** - Encourage students to share their progress on social media or with friends
7. **Pair Programming** - Have students work in pairs to promote collaboration
8. **Regular Breaks** - Take breaks every 45-60 minutes to maintain focus
